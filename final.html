  • Project Description

    Monarchy Madness is a text-based roleplay game that allows you to govern an empire. Players will be given various situations to respond to each year. Try to make the best decisions for the empire!

  • Instructions

    Use the first link to access the game. You will need to download the file and then press "run" to start the game. You may also click the website link to get an in-depth explanation on how to play, development details, and extra visuals.

  • How it Works

    When the player clicks "run," the file loads a welcome message that allows the player to create a name for their character and empire using input. After this, the file loads a menu containing options like "military menu," "province management," "graphs," and the option to access their annual dilemma. The military menu and province management uses global variables and input to generate options. The graphs are created using lists that keep track of changes in status and matplotlib. The annual dilemma is generated using random to access a dictionary of 26 events and 52 possible outcomes. The player inputs either a 0 or 1 to give their response. If the response is not a 0 or 1, it results in an IndexError and the player will have to restart the game.
